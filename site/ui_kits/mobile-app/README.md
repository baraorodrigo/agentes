# Mobile app — UI kit

A click-thru mobile companion for El Coyote Pub. Patrons browse the agenda, buy tickets, and access their entry pass.

**Screens (left → right inside index.html):**
- Home — next event, agenda strip
- Event detail — lineup, lots, buy CTA
- Ticket — entry pass with QR + door code
- Menu — bar / kitchen tabs

**Tech:** React 18 (UMD + Babel) inside the `ios_frame.jsx` starter, four iPhones rendered side-by-side.
