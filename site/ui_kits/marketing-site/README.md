# Marketing site — UI kit

A click-thru marketing site for El Coyote Pub.

**Screens:**
- `home` — hero with rotating photo, tagline, next-up event, agenda strip
- `agenda` — full event listing with filter chips
- `event` — single event detail with sticky buy CTA
- `private` — private events page (weddings, corporate)
- `contact` — find us / map / hours

**How to read:** open `index.html`. Use the left rail to jump between screens. Each screen demonstrates one or more components from the system.

**Tech:** React 18 via UMD + Babel inline. All copy in Brazilian Portuguese.
