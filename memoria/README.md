# Memória — Sub-mapa

Registro histórico do El Coyote. A IA consulta aqui quando precisa de contexto sobre decisões passadas, pessoas ou eventos.

| Arquivo | Conteúdo | Status |
|---------|----------|--------|
| `projetos.md` | Projetos ativos e concluídos | Vazio |
| `decisoes.md` | Decisões do Rodrigo registradas | Vazio |
| `licoes.md` | Lições aprendidas (o que deu certo/errado) | Vazio |
| `pessoas.md` | Promoters, contatos, fornecedores | Vazio |
| `eventos.md` | Histórico de eventos com métricas | Vazio |
| `pendencias.md` | Itens aguardando input do Rodrigo | Deploy na VPS |

## Regras

- Todo arquivo de memória tem data de última atualização no topo
- Dados financeiros só em arquivos marcados como "admin-only"
- Memória não substitui dados frescos — sempre preferir extração atual do PNE/Bar Fácil
